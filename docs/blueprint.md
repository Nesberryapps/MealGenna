# **App Name**: MealMind

## Core Features:

- Home Screen Dashboard: Greet the user by name and provide quick mood selection buttons (Light, Comfort, Quick, Healthy) on the home screen.
- Pantry Management: Allow users to view, add, edit, and delete pantry items with details like name, quantity, unit, and expiry date.
- Intelligent Recipe Suggestions: Fetch recipes using an AI agent based on the selected mood, pantry ingredients, and user dietary preferences. The AI tool should prioritize pantry ingredients in choosing a recipe.
- Recipe Display: Display recipes with name, image, preparation time, and a list of required ingredients.
- Grocery List Auto-Generation: Automatically generate a grocery list based on missing ingredients from suggested recipes.
- Grocery List Management: Enable users to add custom items, mark items as purchased, and delete items from the grocery list.
- User Preferences: Allow users to set dietary choices (Vegan, Vegetarian, Keto, None) and units (Metric or Imperial) in the settings screen.

## Style Guidelines:

- Primary color: Green (#4CAF50) to represent freshness and health.
- Background color: Light gray (#F5F5F5) for a clean, minimalist design.
- Accent color: Orange (#FF9800) for call-to-action buttons and highlights to draw attention.
- Body and headline font: 'PT Sans' sans-serif for a modern and accessible style. 
- Use simple, clear icons to represent different food categories, moods, and actions within the app.
- Use a clean, responsive layout optimized for both mobile and web, with large buttons and intuitive navigation.
- Incorporate subtle animations for loading states and transitions to enhance the user experience without being distracting.
