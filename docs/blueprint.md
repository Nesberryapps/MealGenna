# **App Name**: CulinaZen

## Core Features:

- Random Meal Suggestion: Suggest a meal idea at random from a curated list.
- AI Recipe Generator: Generate recipe ideas based on items scanned or entered from the user's pantry using generative AI as a tool to incorporate user preferences.
- Personalized Meal Plan: Create a personalized 7-day meal plan considering dietary preferences, cuisine preferences, and allergies.
- Preference Selection: Allow users to define their dietary and cuisine preferences.

## Style Guidelines:

- Primary color: Soft Blue (#A0D2EB) to create a calming and fresh feeling.
- Background color: Light Gray (#F0F0F0) to maintain a clean and minimalist appearance.
- Accent color: Pale Lavender (#D0BFFF) to add a subtle, elegant contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif for modern, clean readability.
- Use simple, outlined icons for food categories and dietary options.
- Employ a card-based layout to display meal suggestions and recipes.
- Incorporate smooth transitions when switching between meal plans and recipe views.
