# **App Name**: MealGenius

## Core Features:

- Ingredient Scanner: Scan items in your refrigerator or pantry using the device's camera to identify ingredients.
- Meal Idea Generation: Suggest meal ideas based on the scanned ingredients, using AI to generate recipes or recommend existing ones. AI acts as a tool.
- Manual Ingredient Input: Allow users to manually enter ingredients if scanning is not feasible or desired.
- Quick Meal Ideas: Generate quick meal ideas for users short on time, providing simple and fast recipe options.
- 7-Day Meal Plan: Generate a 7-day meal plan, either from the input ingredients or random, while generating recipes using AI, with proper warnings about user's allergies, and dietary preferences. AI acts as a tool.
- Recipe Display: Display recipes in a clear and user-friendly format, including ingredients, instructions, and preparation time.

## Style Guidelines:

- Primary color: Emerald green (#50C878) to convey freshness and health.
- Background color: Dark navy (#192841) to create contrast with content elements and imply sleekness.
- Accent color: Gold (#FFD700) to highlight calls to action and add premium flair.
- Body and headline font: 'Inter', a sans-serif, provides a modern, machined look.
- Use minimalist, outlined icons related to food and cooking.
- Grid-based layout to provide balance and promote effective scanning.
- Use transition animations, e.g. fading between ingredients.
